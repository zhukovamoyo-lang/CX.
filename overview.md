# Обзор проекта

Цель: одностраничный сайт для презентации направления «Клієнтський досвід» MOYO.

Ключевые функции:
- 6 интерактивных кнопок с заголовками.
- По клику открывается модальное окно с соответствующим текстом.
- Цветовая схема и акценты в стиле логотипа MOYO, логотип увеличен.

Результат: быстрый, адаптивный, доступный сайт без зависимостей.
