# Пользовательский поток и структура

## Поток
1. Пользователь открывает страницу и видит логотип и заголовок.
2. Выбирает одну из 6 кнопок.
3. Открывается модальное окно с подробным текстом.
4. Закрывает модалку кнопкой, кликом по фону или Esc.

## Структура страницы
- Шапка с логотипом.
- Заголовок из двух строк.
- Сетка из 6 плиток-кнопок.
- 6 модальных окон с соответствующими текстами.
